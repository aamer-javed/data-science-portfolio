# Order Picking, Batching, and Routing Optimization

## Objective

Solve warehouse order batching, picking, and routing problems using operations research methods and practical heuristics.

## Optimization Problems

- Assign orders to picking waves
- Minimize total travel distance
- Balance work across robots or pickers
- Meet cutoff/SLA constraints
- Compare exact solvers with faster heuristics

## Methods to Build

- Greedy batching baseline
- OR-Tools optimization model
- Local search improvement
- Simulated annealing or genetic algorithm stretch goal
- KPI comparison across strategies

## Evaluation Metrics

- Total travel distance
- Average route length
- Orders completed per wave
- SLA miss count
- Runtime
- Workload balance

## Business Questions

- Which batching strategy reduces travel time most?
- How much improvement is possible compared with simple FIFO batching?
- What tradeoff exists between optimality and runtime?
- How should picking waves be designed under demand pressure?

## Skills Demonstrated

Operations research, routing, scheduling, OR-Tools, MIP concepts, heuristics, optimization tradeoffs, warehouse operations analytics.
